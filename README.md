## 📋 What lives here

This repository is the **organisation-level `.github` meta repo** for [Hack23 AB](https://github.com/Hack23). Files here apply to **every** Hack23 repository that does not override them locally.

| Path | Purpose |
|:-----|:--------|
| [`profile/README.md`](profile/README.md) | The public **Hack23 organisation profile** rendered on <https://github.com/Hack23> — flagship projects, services, ISMS, blog, contact. |
| [`FUNDING.yml`](FUNDING.yml) | Default **GitHub Sponsors** configuration — adds the 💖 Sponsor button to every Hack23 repo that doesn't override it. |
| [`ARCHITECTURE.md`](ARCHITECTURE.md) | How this org-meta repo is structured and how GitHub renders the org profile. |
| [`SECURITY_ARCHITECTURE.md`](SECURITY_ARCHITECTURE.md) | Defense-in-depth controls for the documentation system (per Hack23 ISMS Secure Development Policy). |
| [`THREAT_MODEL.md`](THREAT_MODEL.md) | STRIDE + MITRE ATT&CK threat model for the public-documentation attack surface. |

> 📚 **Looking for the org profile?** It's at [`profile/README.md`](profile/README.md) and rendered at <https://github.com/Hack23>.
